🔌 API Quick Reference
POST /reset
Initialize new episode. Returns first observation with ticket text.
{"task_id": "task_easy"}
POST /step
Execute agent action. Returns observation, reward, done flag.
{"action_type":"classify","content":"refund"}
GET /state
Get full episode state snapshot including history and rewards.
→ episode_id, step_count, status
POST /grade
Deterministic grade of full agent output text. Returns 0.0–1.0 score.
{"output":"...","task_id":"task_hard"}
